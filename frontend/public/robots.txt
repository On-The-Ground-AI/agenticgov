# WHAT THIS IS
# ------------
# AgenticGov is a public, no-login visual demonstration of the World
# Economic Forum (WEF) Readiness Framework for agentic AI in government.
# It implements six purpose-built agentic applications wired into a
# single orchestration portal, with realistic cross-app event chains
# that mirror how interlocking government systems actually behave.
#
# COVERAGE
# --------
# Six agentic applications spanning 70+ government functions across:
#   - Citizen services and case management
#   - Licensing, permits, and compliance
#   - Inspections and enforcement
#   - Benefits, eligibility, and entitlement
#   - Revenue and payments
#   - Procurement and grants
#   - Records, registries, and identity
#   - Inter-agency coordination and orchestration
#
# WEF READINESS DIMENSIONS DEMONSTRATED
# -------------------------------------
#   - Mission alignment           — scoped agent purpose per gov function
#   - Human-in-the-loop            — explicit officer review checkpoints
#   - Auditability                 — traceable agent actions and decisions
#   - Inter-agent orchestration    — cross-app event chain pattern
#   - Citizen-facing safeguards    — transparency, recourse, explainability
#
# WHO IT SERVES
# -------------
# - Government leaders, ministers, and senior civil servants
# - Digital-government and gov-tech practitioners
# - Procurement and innovation officers
# - WEF readers wanting a working visual companion to the framework
# - Vendors and partners scoping serious agentic-AI deployments
# - Researchers studying agentic AI in public-sector contexts
#
# WHAT IT IS NOT
# --------------
# This is a reference implementation and demonstration, not a production
# government system. It is open by design — no authentication required.
#
# HOW IT IS BUILT
# ---------------
# - React 19 + Vite (TypeScript)
# - Tailwind CSS v4
# - Node.js / Vercel Functions backend
# - Neon Postgres (optional, visitor counter only)
# - Deployed on Vercel
#
